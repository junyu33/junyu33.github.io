我们首先简要介绍一下有理函数和除子 div\text{div}div 的概念：

对于任意单变量有理函数：

f(X)=a0+a1X+a2X2+⋯+anXnb0+b1X+b2X2+⋯+bmXmf(X) = \frac{a_0+a_1 X + a_2 X^2 + \cdots + a_n X^n}{b_0+b_1 X + b_2 X^2 + \cdots + b_m X^m} f(X)=b0​+b1​X+b2​X2+⋯+bm​Xma0​+a1​X+a2​X2+⋯+an​Xn​

根据代数基本定理，我们总可以在复平面上将其因式分解为：

f(X)=a(X−α1)e1(X−α2)e2⋯(X−αr)erb(X−β1)d1(X−β2)d2⋯(X−βs)dsf(X) = \frac{a(X-\alpha_1)^{e_1}(X-\alpha_2)^{e_2} \cdots (X-\alpha_r)^{e_r}}{b(X-\beta_1)^{d_1}(X-\beta_2)^{d_2} \cdots (X-\beta_s)^{d_s}} f(X)=b(X−β1​)d1​(X−β2​)d2​⋯(X−βs​)ds​a(X−α1​)e1​(X−α2​)e2​⋯(X−αr​)er​​

这里 αi,βj\alpha_i,\beta_jαi​,βj​ 共 r+sr+sr+s 个数互不相同。

为了简便，我们可以使用除子将式子简化：

div(f)=e1[α1]+e2[α2]+⋯+er[αr]−d1[β1]−d2[β2]−⋯−dr[βr]\text{div}(f) = e_1[\alpha_1] + e_2[\alpha_2] + \cdots + e_r[\alpha_r] - d_1[\beta_1] - d_2[\beta_2] - \cdots - d_r[\beta_r] div(f)=e1​[α1​]+e2​[α2​]+⋯+er​[αr​]−d1​[β1​]−d2​[β2​]−⋯−dr​[βr​]

这里的αi\alpha_iαi​被称为 zeros（零点），βi\beta_iβi​ 被成为 poles（极点），而前面的系数ei,die_i,d_iei​,di​则被称为 multiplicity（重数）。

当然，这里的 XXX 大写就意味着 fff 的对象不一定只能是一个数，XXX 的本身也可以由多变量的函数构成，因此 fff 也变成了多变量函数。进一步地，如果f=f(x,y)f = f(x,y)f=f(x,y) 中的 (x,y)(x,y)(x,y) 由椭圆曲线 EEE 所约束，那么 EEE 中所对应的点有的可以让分子为零，有的可以让分母为零，因此这些点也可以被分别称为零点和极点。

div(f)=[P1]+[P2]+[P3]−3[O]\text{div}(f) = [P_1] + [P_2] + [P_3] - 3[\mathcal{O}] div(f)=[P1​]+[P2​]+[P3​]−3[O]

经过上述例子我们可以猜想 EEE 上 div(f)\text{div}(f)div(f) 的一些性质：

  1. div(f)=div(g)\text{div}(f)=\text{div}(g)div(f)=div(g)，当且仅当存在常数ccc，f=cgf=cgf=cg.
  2. 展开式系数（度数）之和为000.
  3. 如果对这几个点做点加运算，结果为 O\mathcal{O}O.
  4. 特别地，如果不存在零点和极点，那么 fff 为常数。

前者被称为除子的唯一性定理，后三者被称为主除子定理。结论与有理函数fff的选取无关。

weil pairing

我们设有理函数 fP,fQf_P, f_QfP​,fQ​ 满足 div(fP)=m[P]−m[O],div(fQ)=m[Q]−m[O]\text{div}(f_P) = m[P] - m[\mathcal{O}], \text{div}(f_Q) = m[Q] - m[\mathcal{O}]div(fP​)=m[P]−m[O],div(fQ​)=m[Q]−m[O]。假设P,Q∈E(m)P, Q \in E(m)P,Q∈E(m)，且 SSS 不在P,QP, QP,Q 张成的群里，则定义 weil pairing：

em(P,Q)=fP(Q+S)fP(S)/fQ(P−S)fQ(−S)e_m(P, Q) = \frac{f_P(Q + S)}{f_P(S)} / \frac{f_Q(P - S)}{f_Q(-S)} em​(P,Q)=fP​(S)fP​(Q+S)​/fQ​(−S)fQ​(P−S)​

weil pairing 的值除了与有理函数 fP,fQf_P, f_QfP​,fQ​ 以及 SSS 的选取无关外，还有以下几种性质：

  • 单位根：em(P,Q)m=1e_m(P, Q)^m = 1em​(P,Q)m=1
  • 双线性：em(P1+P2,Q)=em(P1,Q)em(P2,Q)e_m(P_1 + P_2, Q) = e_m(P_1, Q)e_m(P_2, Q)em​(P1​+P2​,Q)=em​(P1​,Q)em​(P2​,Q)，em(P,Q1+Q2)=em(P,Q1)em(P,Q2)e_m(P, Q_1+Q_2) = e_m(P, Q_1)e_m(P, Q_2)em​(P,Q1​+Q2​)=em​(P,Q1​)em​(P,Q2​)
  • 非退化性：∀P∈E(m),  s.t.  em(P,Q)=1⇒Q=O\forall P \in E(m), \; s.t. \; e_m(P, Q) = 1 \Rightarrow Q = \mathcal{O}∀P∈E(m),s.t.em​(P,Q)=1⇒Q=O
  • 交错性：em(P,Q)=em(Q,P)−1e_m(P, Q)=e_m(Q, P)^{-1}em​(P,Q)=em​(Q,P)−1, em(P,P)=1e_m(P, P) = 1em​(P,P)=1.
