(sticky) Meta Information

Lists some meta information related to the blog itself for easy reference.

Architecture:

  • Hexo + NexT (Mist) Hexo + World
  • GitHub Pages for static page hosting
  • Netlify for CDN and IPv6 access support

Plugins:

  • Twikoo for comments (very rarely used, deprecated)
  • Aplayer + MeingJS for music (very rarely used, deprecated)
  • Busuanzi for traffic statistics (Google Search Console is sufficient)

Version Changelog:

  • 2021/07/06: Build completed.
  • 2022/07/26: Changed domain from junyu33.github.io to blog.junyu33.me.
  • 2023/01/30: Upgraded hexo, nexT, and twikoo plugins.
  • 2024/11/18: Added a dark theme, removed background images, and fixed the display issue with Busuanzi.
  • 2025/08/12: Changed the theme to "world", added i18n (spent 2 weeks translating original posts), moved the image hosting from the local server to img.junyu33.me, and removed the dark theme and Busuanzi.
  • 2026/01/30: Archived all the problem solutions I wrote on Luogu during high school (including those that were not approved) to my blog.
  • 2026/04/25: Due to Netlify’s pricing policy change on 2025/09/04 and the increased deployment cost under the new credit-based model, this blog has been migrated from Netlify to Vercel. As a trade-off, the blog no longer provides native IPv6 access.
  • 2026/05/03: Add a mirror for China Mainland visitors.
