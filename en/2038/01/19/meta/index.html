(Sticky) Meta Information

Lists some meta information related to the blog itself for easy reference.

Architecture:

  • Hexo + NexT (Mist) Hexo + World
  • GitHub Pages for static page hosting
  • Netlify for CDN and IPv6 access support

Plugins:

  • Twikoo for comments (very rarely used, deprecated)
  • Aplayer + MeingJS for music (very rarely used, deprecated)
  • Busuanzi for traffic statistics (Google Search Console is sufficient)

Version Changelog:

  • July 6, 2021: Build completed
  • July 26, 2022: Changed domain from junyu33.github.io to blog.junyu33.me
  • January 30, 2023: Upgraded hexo, nexT, and twikoo plugins
  • 2024/11/18: Added a dark theme, removed background images, and fixed the display issue with Busuanzi.
  • 2025/08/12: Changed the theme to "world", added i18n, moved the image hosting from the local server to img.junyu33.me, and removed the dark theme and Busuanzi.
