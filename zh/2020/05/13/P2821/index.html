题解 P2821 【变幻数】

既然有高精，怎么能没有python呢？

n=input()
n=int(n)
if n<10:print(n) #特判
else:
   i=9
   flg=0
   st=''
   while 1:
      if n==1: break
      if i==1:
         flg=1 
         print("There is no such number!") 
         break
      if n%i==0:
         st+=str(i)
         n//=i #这里一定要用整除，不然n的类型会变成double，从而导致精度炸掉
      else: i-=1
   if flg==0: print(st[::-1]) #倒序
